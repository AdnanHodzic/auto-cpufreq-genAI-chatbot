When you install `auto-cpufreq` using snap package, you can check daemon status using: ``` systemctl status snap.auto-cpufreq.service.service ``` Also there's just been released a [major new release](https://github.com/AdnanHodzic/auto-cpufreq/releases/tag/1.2), so make sure to upgrade :) Closing the issue, please feel free to re-open it if it doesn't address your question.

How are distros supposed to deal with the log file?

Issue #146

Description

I'm creating a NixOS module at NixOS/nixpkgs#109065, and the topic of the log file came up. The problem is that there is no automatic way to clear the file and the file grows by 2KB every 5 seconds. Also the log file has to exist before hand, otherwise the service doesn't start. We are confused about how do deal with the log file since it continually grows in size and there isn't an automatic way to clear it.

Let's continue discussion from #59 here. All you need to do to build a snap with current structure is: ``` snapcraft ``` After build is successful, all you need to do is install the snap, i.e: ``` sudo snap install auto-cpufreq_1.2_amd64.snap --dangerous ``` and then enable necessary interfaces, i.e: ``` sudo snap connect auto-cpufreq:cpu-control sudo snap connect auto-cpufreq:hardware-observe sudo snap connect auto-cpufreq:system-observe ``` A lot of people on on forum.snapcraft.com are Canonical employees and are more then willing to help as you [can see from an example](https://forum.snapcraft.io/t/snap-not-buildable-anymore/19274). For everything else [I suggest you refer to their official documentation](https://snapcraft.io/docs/creating-a-snap) or a ask me a question here :)

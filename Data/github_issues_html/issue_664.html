It's because you're using `acpi-cpufreq` as your scaling driver which doesn't make use of EPP. To learn more about the differences between `acpi-cpufreq`, `amd-pstate`, and `amd-pstate-epp`, visit the kernel docs [here](https://docs.kernel.org/admin-guide/pm/amd-pstate.html#amd-pstate-vs-acpi-cpufreq)

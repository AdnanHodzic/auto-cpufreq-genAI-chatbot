[Feature] Power Governer Configurable Depending on Battery Life

Issue #356

Description

The power governor is currently only configurable depending on if the device is charging or discharging. Have the ability to change the power governor based on current battery life. Example use case: I want to charge quickly, once I charge to +90% use performance governor instead 0%-90% = Powersave 90%-100% = Performance

[feature request/enhancement] settings file

Issue #26

Description

Would it be a good idea to have a settings file? Certain things could be stored here like; - minimum CPU load before going to turbo - minimum load factor before going to turbo - true/false for allowing turboboost to be switched off when on battery And any other items that can be changed.

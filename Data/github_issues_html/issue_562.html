Is the frequency in the config file the base clock or is it the effective clock frequency?

Issue #562

Description

There is no error per say but I wanted to have clarification on what the frequency in the config file represents. I'm using an i7-11800H. For this chip, the base clock can go up to 2.3GHz, but after the multiplier is applied, the net clock can go much higher (around 4.8GHz).

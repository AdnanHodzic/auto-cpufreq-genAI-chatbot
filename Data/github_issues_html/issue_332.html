Improve charging detection function

Issue #332

Description

The list of items in power_supply is now sorted. This way the AC is 'always' on top, and checked first. What I noticed is that internal supplies are always use uppercase, and inserted devices all use lowercase. So if this theory is correct the AC and internal battery will always be checked first. I also noticed the "scope" file under the hid device with battery. Maybe that file could help identify if an battery is from some other device or not. Will see in future bug reports if that is correct and maybe use that. If this still not works correctly I've implement an ignorelist. Where we can easily add items or parts of the string from devices we want to be ignored. Currently in the ignore list is the "hidpp_battery" that came up in every issue so far. It's just an improvement of what we already have. Fixes #331

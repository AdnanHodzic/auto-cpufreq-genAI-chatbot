[FR] Allow 2 different governors (without power supply & with power supply) & automatically switch between them

Issue #879

Description

Hi, One generally needs to use 2 different governors: 1) **with power supply** connected one e.g. likely needs `Balanced` or `Performance`. 2) **Without power supply** connected one likely needs `Powersave`. So basically **every time power supply connected & disconnected** one has to manually switch the governor. It would be great if auto-cpufreq could detect if a power supply is connected and automatically switch between these modes. Thanks.

Comments

You can configure this with [auto-cpufreq config file](https://github.com/AdnanHodzic/auto-cpufreq/?tab=readme-ov-file#4-auto-cpufreq-config-file).

Description

Hi there, auto-cpufreq has been great for me to strangle my 5800HC CPU and stop it cranking up and wasting battery I've got a suggestion/improvement I'd like you to consider, it's stated that cpu-autofreq uses and works best with the "balanced" power profile presumably referencing the setting in /sys/firmware/acpi/platform_profile Based on some benchmarks done by phoronix etc, battery life would be better with the low-power profile as in most cases this tells the laptops BIOS to conserve power, leveraging not software or drivers but actual firmware features to save power. If this profile switcher was incorporated into cpu-autofreq's config so that for example, on battery low-power profile is set and on AC this is changed back to balanced, I think we'd see some further gains in power saving. Is this something you could look into? example source: https://www.phoronix.com/review/ryzen7-6850u-acpi/6

    • "That's good and all, but is there a way to have it on powersave all the time except if there's a lot of work to do then have it go to performance just until the work is done?" Yes, that's exactly what auto-cpufreq will do by "default", if it's in powersave, there's a lot of work, it'll switch to performance and turn on turbo boost, once all the heavy work is finished it'll go back to powersave. *But*, if you want to keep it in powersave all the time, you could do this using config file. Set it to be always on powersave and enable "turbo_boost", I explained more in v2.0 video: https://www.youtube.com/watch?v=SPGpkZ0AZVU&t=608s

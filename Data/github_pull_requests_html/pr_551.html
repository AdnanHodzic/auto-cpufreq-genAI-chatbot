Description

Some Distros do not have /bin/bash by default, but have bash installed at some other location and in PATH. To allow all of these distros to use these shell scripts, change the shebangs to search for bash in PATH. More to why this is better [here](https://unix.stackexchange.com/questions/29608/why-is-it-better-to-use-usr-bin-env-name-instead-of-path-to-name-as-my)

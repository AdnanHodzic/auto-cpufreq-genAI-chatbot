Implement support for Fedora 41

Pull Request #786

Description

This PR closes #779. It adds the support to Fedora 41. Fedora 41 uses [TuneD](https://docs.fedoraproject.org/en-US/fedora/latest/release-notes/sysadmin/#tuned-def-pwr-mgmt-daemon) as a default power profile management daemon and it is conflicting with auto-cpufreq service. Thanks to this PR, TuneD is automatically disabled when installing auto-cpufreq.

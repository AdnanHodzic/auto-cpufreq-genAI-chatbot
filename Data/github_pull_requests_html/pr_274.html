On laptops and notebooks, check if charging via BAT* status

Pull Request #274

Description

Hi, For some time now auto-cpufreq got stuck on the powersave governor for me. This happens because Linux does not recognize my power adapter for some reason, thus using `grep . /sys/class/power_supply/A*/online` doesn't work for me. I discussed this further in #234. So, I changed `charging()` to read from `/sys/class/power_supply/BAT*/status` instead, which will greatly improve my experience.
